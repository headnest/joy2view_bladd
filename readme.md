# joy2view

Blender向けの、Direcrt inputデバイスを視点操作に利用するためのアドオン

## 機能

direct inputデバイスのスティック各軸の入力をblender内の視点の回転(orbit)、並行移動、ズーム/前後にマッピングします

[N] キーを押してプロパティパネル(3Dビューエリア右上)を表示し、アドオンのタブ内の開始ボタンを押せば動作状態になりジョイスティック入力が有効になります。
現在の実装ではワークスペースを跨いでは動作できないので、切り替えのたびにオンオフを行ってください。

現在は作者所有のデバイスでのレイアウトで割当を行っていますが、将来的にはプロパティパネル内でマッピングを変更可能にする予定です



## 導入方法

1. zipファイルとしてダウンロード(解凍不要)
2. blender内のpreferenceからアドオン管理画面へ
3. ファイルからインストールを選択、1で選択したzipファイルを選択
4. 有効化する
